# Parcial 2 - JavaScript DOM

Estructura de entrega sugerida:

```txt
Apellido_Nombre_Parcial_2/
  ejercicio_01/
    index.html
    style.css
    script.js
  ...
```

Cada ejercicio incluye el HTML base, CSS básico opcional y el archivo `script.js` con el enunciado comentado.

## Ejercicios

1. **Cambiar texto de un elemento** — DOM básico + evento click
2. **Validar input vacío** — Inputs + condicionales
3. **Agregar elementos a una lista** — Creación de elementos + listas
4. **Recorrer array y mostrarlo en pantalla** — Arrays + renderizado
5. **Contador con botones** — Eventos + variables + actualización del DOM
6. **Detectar cantidad de caracteres** — Eventos de teclado + strings
7. **Filtrar nombres de un array** — Arrays + strings + filtro dinámico
8. **Cambiar estilos desde JavaScript** — Manipulación de estilos
9. **Mostrar solo números pares** — Arrays + lógica condicional + DOM
10. **Convertir texto a mayúsculas** — Strings + input + DOM
11. **Validación simple de formulario** — Formulario + validaciones + submit
12. **Buscar Pokémon por nombre usando PokeAPI** — Fetch + petición GET + JSON + renderizado en DOM
